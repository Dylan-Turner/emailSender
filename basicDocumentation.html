Setup

To setup please sign up and complete your registration to Form Sender.

1. Set up the HTML form

Set up a HTML form, include as many input, textarea and other form tags as you want. Make sure that you set your HTML form's action to go to the correct server adress as such: 

<form action="https://formsender.000webhostapp.com/server.php" method="POST" id="form">
