FAQ

Here are some of the most frequently asked questions, hopefully you can find the answer to any question you have here.

  1. The email hasn't been sent!

    2. If the email hasn't been sent and you have copied the example code and changed all of the attributes to be what you want them to be ensure that you have signed up. You must sign up with the same email that you are sending the emails to. If the emails still are not sending, please make sure that you have set everything up correctly and if it still doesnt work contact me.

  2. I have run into an error when after the email has sent!

    3. If you run into any errors try looking at the basic documentation or check out the troubleshooting guide there. There are lots of little things you could have missed such as naming your tags or giving values to ceratin things so make sure to check the troubleshooting guide out! (the troubleshooting guide is on the basic documentation page)

  3. Do you keep the emails I send?

    4. We Do NOT! All emails are removed every minute!

  4. Someone is sending me spam emails!

    5. If someone is sending spam or innapropiate and unwanted emails to you please contact me. If you would like to be put on a Do Not Send list also contact me and you will never recieve and email through us again.

  5. I have another problem

    6. If you have another problem, please contact me. Please note that I may take up to a couple of days to reply.
