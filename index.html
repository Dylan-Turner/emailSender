Basic Usage

To use Form Sender simply make your HTML form with as many elements as you want. You can include inputs, textareas, selects and more. Once you have created your HTML form you need to add two hidden inputs, one for the email adress you would like the form to go to and one for the page that you want to re-direct to after the form has been successfully submitted. You then need to ensure that the form's action is to the correct adress.
Example: 

                <form action="[ServerLink]" method="POST" id="form">

                  <input class="hidden" type="text" name="EMAILTO" value="ex@mple.com">
                  <input class="hidden" type="text" name="GOTO" value="https://dylanturner.tech">

                  <input type="text" name="name" required>
                  <input type="text" name="email" required>
                  <textarea name="comments" rows="5" cols="50"></textarea>

                  <input type="submit" value="Submit results" id="submitButton">

                </form>

You can have as many inputs as you want. If you are interested in more advanced usage, go see the documentation.
